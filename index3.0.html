Questionário
Questão 1 : Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

R: entrega arquivos estáticos, o que está pronto como o HTML e imagens.
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

R: É a sua alta performance e ele é muito muito utilizado em projetos de alto desempenho pois ele consegue lidar com milhares de conexões simultâneas sem sobrecarregar o servidor.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

R: O HTTP é um protocolo para transferência de páginas web sem criptografia, ou seja, sem segurança. Já o HTTPS é a versão do HTTP com segurança (criptografia).
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

R: São canais de comunicação que permitem que diferentes serviços e aplicações se comuniquem simultaneamente usando o mesmo endereço IP. A importância da porta 443 é a criptografia.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

R: É o número identificador de um dispositivo na rede, ele identifica e localiza um dispositivo na rede.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

R: É um servidor virtual com mais controle e desempenho, onde um servidor físico é dividido em várias máquinas virtuais isoladas, cada uma com seu próprio sistema operacional, memória, CPU e armazenamento dedicados. É indica em situações que exigem mais controle, segurança e escalabilidade.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

R: O Caddy tem um servidor moderno HTTPS automático, configuração simples enquanto os outros servidores o HTTPS é automático, a configuração mais complexa.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

R: O Kestrel, por ter um desenvolvimento alto, ser multiplataforma, é integrado ao ASP.NET.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

R: É o modelo onde o cliente envia requisições a um servidor, que responde com os dados/processamento solicitado.
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

R: É um servidor de aplicações de código aberto, utilizado principalmente para executar aplicações Java.
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

R: O servidor web entrega conteúdo, enquanto o servidor de aplicação executa o código da aplicação para gerar esse conteúdo.
Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

R: Um domínio é como o nome de um site que a gente digita no navegador, tipo google.com. Ele serve pra gente não precisar decorar números complicados, que são os endereços IP, que é onde o site realmente está hospedado.
Questão 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

R: Escalabilidade fácil e rápida, alta disponibilidade e redundância, redução de custos com infraestrutura, acesso remoto de qualquer lugar.
Questão 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

R:Apache Tomcat e GlassFish com Java.
Questão 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

R:A utilização do HTTPS é essencial pois possui criptografia, o HTTP apresenta riscos de segurança por não possuir criptografia, podendo ter vírus no site sem criptografia.
Questão 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

R: Usar VPS vale muito mais a pena do que uma hospedagem compartilhada, porque dá mais controle e não deixa na mão quando o site começa a crescer, por exemplo quando se tem uma loja online.
Questão 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

R:O IP mostra onde está o dispositivo na rede, e a porta mostra o que exatamente você quer acessar dentro dele.
Questão 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

R:Um servidor web que aceita HTTPS é super importante porque protege os dados, transmite confiança para o usuário e até ajuda no Google.
Questão 19: Cite dois exemplos de servidores web e explique suas principais características.

R: Apache HTTP Server possui código aberto como uma de suas características e o NGINX que é ótimo para alto desempenho e proxy reverso.
Questão 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

R: A hospedagem compartilhada é quando vários sites dividem o mesmo servidor. É uma boa opção pra começar na internet sem complicação e sem gastar muito.
Questão 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

R: O servidor de aplicação é o que realmente roda o sistema, faz cálculos, responde a formulários, acessa o banco, ele pensa antes de responder. Já o servidor de arquivos estáticos só entrega arquivos prontos, como se fosse uma pasta compartilhada na internet.
Questão 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

R: O Gunicorn é um servidor que roda aplicações Python na web. Ele recebe as requisições, chama o código da sua aplicação, e devolve a resposta. É leve, rápido e muito usado com Flask ou Django, geralmente junto com o NGINX em produção.
Questão 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

R: É uma plataforma corporativa da Red Hat, é um servidor de aplicação feito para rodar sistemas grandes em Java. Ele é perfeito pra projetos corporativos complexos, porque já vem com tudo pronto: banco de dados.
Questão 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

R: A porta 21 é usada para conectar e controlar a comunicação entre cliente e servidor no protocolo FTP, que serve para enviar e receber arquivos pela rede.
Questão 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

R: O LiteSpeed é compatível com o Apache porque entende as mesmas regras e arquivos de configuração, o desempenho é melhor e mais rápido.
Questão 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

R: O IIS é um servidor para Windows que ajuda a rodar sites e aplicativos feitos com tecnologias Microsoft. É uma escolha comum para empresas que querem algo simples de usar e que funcione bem com o que já usam no Windows.
Questão 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

R: O Node.js serve pra rodar JavaScript no servidor e é ótimo para criar sites e apps rápidos, principalmente aqueles que precisam funcionar em tempo real ou atender muita gente sem travar. É muito usado para APIs, chats, jogos e sistemas que crescem rápido.
Questão 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

R: O GlassFish é um servidor que roda sistemas feitos em Java, principalmente aqueles grandes, corporativos.
Questão 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

R: Puma e Unicorn são servidores que fazem aplicações feitas em Ruby on Rails funcionarem na web. Ambos são muito usados em projetos Ruby.
Questão 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

R: Escolher o tipo certo de hospedagem é fundamental para o projeto funcionar bem, não gastar demais e crescer sem dor de cabeça. Se escolher errado, pode ter site lento, indisponível ou custo alto demais. Por isso, pensar nisso no planejamento evita muitos problemas depois.
